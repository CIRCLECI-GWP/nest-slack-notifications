## Set up instructions.

### 1. Clone app and install dependencies.

```bash
# Clone the repository
git clone https://github.com/CIRCLECI-GWP/nest-slack-notifications.git

# CD into the directory
cd nest-slack-notifications

# Install dependencies
npm install
```

### 2. Running tests

```bash
npm run test
```
